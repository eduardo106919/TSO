
# Tecnologias de Sistemas Operativos

Conteúdo abordado na unidade curricular **Tecnologias de Sistemas Operativos** no ano letivo 2025/2026.


## Conteúdo

Neste repositório poderás encontrar o seguinte conteúdo:
- resolução das [fichas](fichas/) práticas
- resolução de [testes](testes/) e exames de anos anteriores
- [slides](slides/) com a matéria abordada na cadeira
- enunciados de [trabalhos](trabalhos/) práticos de vários anos


## Ambiente Laboratorial

Preparação do ambiente laboratorial necessário.

**Primeiro passo**: instalar o software **VirtualBox v7.2.4** - [Virtual Box](https://www.virtualbox.org/wiki/Downloads).
> :warning: Instalar versão especificada.

**Segundo passo**: instalar a **ISO do Ubuntu Desktop 25.04**. Tenham em atenção a versão do Ubuntu e o tipo de ISO que descarregam:
- Para processadores **Intel** e **AMD**, devem descarregar a ISO `"64-bit PC (AMD64) desktop image"` - [plucky](https://releases.ubuntu.com/plucky/).
- Para processadores **ARM**, devem descarregar a ISO `"64-bit ARM (ARMv8/AArch64) desktop image"` - [plucky](https://cdimage.ubuntu.com/releases/plucky/release/).
